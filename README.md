# python_pathlib

Python の pathlib についての紹介するためのリポジトリ

以下に資料があります
https://github.com/yamap55/python_pathlib/blob/main/python_pathlib/note.ipynb

jupyter を動かしたい場合には、devcontainer の設定をしていますので、VS Code と Docker、Git さえあれば各種開発用設定が行われた Python の開発環境が構築され実行が可能です。

## 環境詳細

- Python : 3.11

### 環境構築

1. VS Code 起動
2. 左下のアイコンクリック
3. 「Dev Containers: Reopen in Container」クリック
4. しばらく待つ
   - 初回の場合コンテナー image の取得や作成が行われる
5. 起動したら開発可能
